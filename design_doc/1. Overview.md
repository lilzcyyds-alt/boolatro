# 1. Overview

## Goal

Port the Unity prototype **Boolatro** gameplay loop into a Flutter codebase that follows the existing **cruising** project style.

核心玩法：
- 类 Balatro 的 run structure（Start / Blind / Shop / …）
- **每一个盲注（Blind）是一道逻辑证明题**：玩家构造 proof，通过验证并结算得分，达标后进入结算与商店。

## Non-goals (for early phases)

- UI/动画不追求与 Unity 1:1
- 暂不实现复杂拖拽连接编辑器（后续可迭代）

## Milestones

- ✅ Phase 0: Flutter skeleton + repo
- ✅ Phase 1: cruising-style RunState + Ticker loop + minimal phase flow UI
- ✅ Phase 2: ProofCore (task generator / validator / scoring) pure-Dart port + unit tests
- ⏳ Phase 3: Proof UI (simplified editor)
- ⏳ Phase 4: Shop + Special Cards (incremental)
