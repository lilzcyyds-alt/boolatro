# 📊 Boolatro 分数计算指南

## 🎯 核心公式

```
最终分数 = Chips × Multiplier
```

---

## 📐 计算顺序

### 步骤 1：计算 Chips（筹码）

```
总 Chips = 卡牌 Chips + 规则 Chips + 升级加成 + Partner 卡加成
```

**详细说明**：
1. **卡牌 Chips**：所有打出卡牌的 Chips 相加
2. **规则 Chips**：结论行（最后一行）使用的推理规则提供的 Chips
3. **升级加成**：Alchemy 卡对该规则的 Chips 升级
4. **Partner 卡加成**：
   - **Amper_Stand**: +50 Chips × 使用 `&` 规则的次数
   - **Inversion**: +60 Chips × 使用 `~` 规则的次数
   - **Disco_Process**: +30 Chips × 剩余换牌次数

### 步骤 2：计算 Multiplier（倍数）

```
总 Multiplier = 1 + 规则倍数 + 升级加成 + Partner 卡加成
```

**详细说明**：
1. **基础倍数**：固定为 1
2. **规则倍数**：结论行使用的推理规则提供的倍数加成
3. **升级加成**：Alchemy 卡对该规则的倍数升级
4. **Partner 卡加成**：
   - **Junior_Associate**: +4 倍数

⚠️ **重要**：所有倍数都是**相加**，不是相乘！

### 步骤 3：计算最终分数

```
最终分数 = 总 Chips × 总 Multiplier
```

---

## 📊 计算实例

### 场景设置
- 打出 3 张卡，每张 10 Chips
- 结论使用 **&Elim** 规则（已用 Separatio 升级）
- 拥有 Partner 卡：**Junior_Associate** 和 **Amper_Stand**
- 证明中使用了 1 次 `&` 规则

### 计算过程

#### 步骤 1：计算 Chips

| 来源 | 基础值 | 升级/加成 | 小计 |
|------|--------|----------|------|
| 卡牌 Chips | 10 × 3 | - | 30 |
| &Elim 规则 | 40 | +20 (Separatio) | 60 |
| Amper_Stand | - | +50 (1个&规则) | 50 |
| **总 Chips** | | | **140** |

#### 步骤 2：计算 Multiplier

| 来源 | 基础值 | 升级/加成 | 小计 |
|------|--------|----------|------|
| 基础倍数 | 1 | - | 1 |
| &Elim 规则 | +4 | +2 (Separatio) | +6 |
| Junior_Associate | - | +4 | +4 |
| **总 Multiplier** | | | **11** |

计算：`1 + 6 + 4 = 11`

#### 步骤 3：最终分数

```
最终分数 = 140 Chips × 11 Multiplier = 1540 分 ⭐
```

## ❓ 常见问题

**Q: 证明的行数影响分数吗？**
A: 不影响。只有最后一行（结论）的规则参与计分。

**Q: 倍数是乘法还是加法？**
A: **加法**。所有倍数加成都累加到基础倍数 1 上。

**Q: 升级是永久的吗？**
A: 是的。Alchemy 卡升级会保存在游戏进度中。
